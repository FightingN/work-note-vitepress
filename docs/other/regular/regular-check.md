## 密码校验规则

- 12~20 位的密码，需包含大小写字母、数字和特殊字符。特殊字符包含"@#$&-+.?\_"
- 举例：123456789Aa@

```js
/^(?=.*?[A-Z])(?=.*?[a-z])(?=.*?[0-9])(?=.*?[@#$&+.\-?_])[A-Za-z0-9@#$&+.\-?_]{12,20}$/;
```

- 8-16 位包含字母和数字的密码
- 举例：123456789aaa

```js
/^(?![0-9]+$)(?![a-zA-Z]+$)[a-zA-Z0-9]{8,16}$/;
```

## 手机号校验规则

- 11 位手机号校验
- 举例：13000000000

```js
/^1[3|4|5|6|7|8|9][0-9]\d{8}$/;
```

## 汉字、字母、数字校验规则

- 只允许填写汉字、字母、数字
- 举例：啦啦123abc

```js
/^[a-zA-Z0-9\u4e00-\u9fa5]+$/;
```
